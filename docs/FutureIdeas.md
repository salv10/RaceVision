# Future Ideas

## Session Setup

### Description

Introduce a dedicated Setup entity associated with a Session.

The setup should not belong to the Vehicle because a vehicle can be configured differently for each session.

### Possible attributes

- Front wing
- Rear wing
- Tire pressure
- Camber
- Toe
- Suspension
- Ride height
- Differential
- Brake bias

### Possible future features

- Compare setups
- AI setup recommendations
- Setup templates
- Dry/Wet presets
- Export and import setups
